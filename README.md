\# 🛡 AI-Based Comment Moderation System



An AI-powered system that extracts YouTube comments, classifies them into Hate / Abusive / Toxic / Safe categories, and supports human-in-the-loop moderation.



\## 🚀 Features



\- Live YouTube comment extraction

\- Transformer-based toxicity detection

\- Configurable threshold logic

\- Human-in-the-loop moderation

\- CSV report generation

\- Visualization dashboard

\- Streamlit Web Interface



\## 🛠 Tech Stack



\- Python

\- Streamlit

\- HuggingFace Transformers (toxic-bert)

\- PyTorch

\- YouTube Data API

\- Pandas



\## 🧠 Key Idea



AI assists moderation decisions while keeping humans in control.
